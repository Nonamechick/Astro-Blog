## **ðŸš€ Getting Started**  

### **Prerequisites**  
- Node.js v18+  
- npm/pnpm/yarn  

### **Installation**  
1. Clone the repository:  
   ```bash
   git clone https://github.com/your-username/astro-blog.git
   cd astro-blog
   ```

2. Install dependencies:  
   ```bash
   npm install
   # or
   pnpm install
   # or
   yarn install
   ```

3. Start development server:  
   ```bash
   npm run dev
   ```

4. Open in browser:  
   ```
   http://localhost:4321
   ```
